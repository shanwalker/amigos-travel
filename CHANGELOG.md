# Travel Amigo - Changelog

## Version 2.0 (2026-01-12)

### Changes from v1
- **Hero Section Redesign**: Replaced search box with Journaway-style "Upcoming Trips" horizontal slider
- **Trip Cards**: Premium vertical cards with destination images, duration, price, and spots remaining
- **Navigation Arrows**: Glassmorphism prev/next buttons for trip slider
- **New Trip Images**: Thailand, Vietnam, Bali, Japan destination photography

### Features
- **Cinematic Hero Section**: Full-bleed video background with trip cards slider
- **Glassmorphism Design**: Premium frosted glass cards throughout the site
- **Navbar**: Sticky glassmorphism navigation with logo and magnetic "Book Now" button
- **The Amigo Way Section**: 3D scroll-triggered card animations showcasing Fixed Groups, Vote, and Open Tribe
- **Memory Reel**: Infinite horizontal scrolling gallery with hover effects and full-screen viewer
- **Marker-Pen Testimonials**: Animated orange highlight sweeps on key emotional words
- **Premium Footer**: Deep navy footer with animated social icons
- **Chat FAB**: Floating chatbot button with pulsing notification and glassmorphism chat window

### Design System
- **Primary Color**: Deep Navy (#0A2540)
- **Accent Color**: Amigo Orange (#FFB400)
- **Typography**: Cormorant Garamond (headings) + Outfit (body) - Modern Premium style
- **Effects**: Glassmorphism, magnetic buttons, smooth scroll, typing animations

### Tech Stack
- React + TypeScript + Vite
- Tailwind CSS
- Framer Motion for animations
